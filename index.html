Mission

매장의 전 과정 문제를 해결해
가치 있는 매장을 만들도록 돕는다

예비창업부터 운영, 성장, 폐업 방어까지 — 외식 사업자의 경영 전 과정에서 데이터 기반의 올바른 의사결정을 지원합니다.

Vision

사업자 데이터를 자산화해
운영·유통·금융으로 확장한다

매장 운영 데이터를 OpenRun Score로 자산화하고, 유통 최적화와 금융 서비스(대안신용평가)로 연결해 자영업 생태계 전체를 혁신합니다.

팀 소개
Founder-Market Fit — 현장의 페인포인트 × 데이터·금융 아키텍처. 가장 깊숙한 '현장의 언어'를 '데이터의 언어'로 번역하는 유일한 팀.

THE OPENRUN AI

구독료를 내고 도구를 쓰는 것이 아니라, AI 직원을 고용하는 개념입니다.

DINING LOGIC ENGINE 1.0

THE OPENRUN AI의 사고 로직을 지탱하는 브레인이 Dining Logic 1.0입니다. Claude Sonnet이 GPT와 다르듯, Dining Logic 1.0은 더 오픈런 AI를 구동하는 독자적 알고리즘 엔진입니다. 단순한 앱이 아닌, 사장님을 대신해 판단하고 실행하는 AI 직원입니다.

SaaS 레이어플랫폼 & 데이터

통합 대시보드

매출·리뷰·광고 효율·재고를 한 화면에서 실시간 확인. 파편화된 데이터를 하나로 통합합니다.

오픈런 스코어

운영 데이터를 비재무 신용등급으로 자동 산출. 금융 소외 자영업자의 데이터 자산화를 실현합니다.

플랫폼 연동

배달앱·네이버·카카오 + 카드매출·POS 매출 데이터 자동 수집·통합.

AI 에이전트 레이어자율 실행

자율 광고 집행

날씨·시간·재고 데이터 기반으로 AI가 광고를 스스로 집행. 사장님 승인 한 번으로 완료됩니다.

리뷰 자동 응대

감성 분석 후 맥락에 맞는 답글을 AI가 자동 작성. 브랜드 톤과 일관성을 유지합니다.

발주 자동 추천

매출 예측 기반 식자재 발주량을 AI가 제안·실행. 재고 손실과 과잉 발주를 최소화합니다.
